התראות לטלפון
TRADINGVIEW → TELEGRAM
קבל התראה לטלפון ברגע שהאינדיקטורים שלך מסתדרים — בחינם, בלי שרת, בלי קוד. TradingView שולח ישירות ל-Telegram. אתה מקבל הודעה, בודק, ומבצע ידנית ב-Tradovate.

⏱️ ההקמה לוקחת ~10 דקות, פעם אחת.
⚠️ דרישה אחת: webhook ב-TradingView עובד רק עם תוכנית Pro ומעלה. בחינמית זה לא יפעל.
1
צור בוט טלגרם
  1. פתח טלגרם, חפש @BotFather (עם הוי הכחול)
  2. שלח /newbot
  3. תן שם לבוט (למשל AXIOM Alerts)
  4. תן username שמסתיים ב-bot (למשל axiom_mnq_bot)
  5. BotFather יחזיר לך Bot Token — מחרוזת ארוכה. שמור אותה.
📲 פתח את BotFather
2
קבל את ה-Chat ID שלך
  1. שלח הודעה כלשהי לבוט החדש שיצרת (לחץ Start קודם)
  2. חפש בטלגרם @userinfobot ושלח לו /start
  3. הוא יחזיר לך את ה-Chat ID שלך (מספר, למשל 123456789)
  4. שמור גם אותו.
📲 פתח את userinfobot
⚙️ מחולל ה-Webhook שלך
הכנס את ה-Token וה-Chat ID, ואני אצור לך את כתובת ה-webhook ואת הודעת ההתראה המוכנה.
📍 כתובת WEBHOOK — הדבק ב-TradingView (שדה Webhook URL)
💬 הודעת LONG — הדבק בשדה Message של התראת קנייה
💬 הודעת SHORT — הדבק בשדה Message של התראת מכירה
4
חבר ל-TradingView
  1. פתח את הגרף של MNQ ב-TradingView
  2. לחץ על כפתור ה-Alert (⏰) למעלה
  3. ב-Condition בחר את האינדיקטור שלך (Champion Trend / WealthSignal) או תנאי מחיר
  4. גלול ל-Notifications → סמן Webhook URL
  5. הדבק את כתובת ה-Webhook שיצרת למעלה
  6. בלשונית Message — הדבק את הודעת LONG או SHORT המתאימה
  7. לחץ Create. זהו!
💡 צור התראה נפרדת לכל כיוון: אחת לכניסת LONG (עם הודעת LONG) ואחת ל-SHORT.
5
בדוק שזה עובד
  1. ב-TradingView, צור התראת בדיקה זמנית (תנאי שקורה מיד, כמו "price crosses" קרוב למחיר הנוכחי)
  2. תוך שניות אמורה להגיע הודעה לטלגרם שלך
  3. אם הגיעה — הכל מחובר. מחק את התראת הבדיקה.
  4. אם לא — בדוק: Token נכון? Chat ID נכון? לחצת Start על הבוט?
✅ מעכשיו, בכל פעם שהאינדיקטור שלך נותן אות — תקבל התראה מיידית לטלפון. בלי לשבת מול המסך.
🔒 חשוב: ההתראה רק מודיעה לך. היא לא מבצעת עסקה. אתה פותח Tradovate, בודק את האות, ומבצע ידנית — בדיוק כמו ש-Apex דורש.
